About Me

So what should I tell you? I’m a passionate Developer and Designer. Programming is what connects me with people and where I learn every day something new.

It all started when I was around 12 years old when my teacher showed us Scratch. A platform where you can build simple games with code blocks. Since then I wanted more and more. So first I learned Python and created some games with pygame and also discord bots. With the years I learned web development so HTML and CSS and created a simple online store (of course without payments).

A game I play since my childhood is Minecraft. And in the time I learned programming I played it, so I wanted to create my own little plugins. The problem was I had to learn java. But with the time my knowledge about java grew and so I created also plugins for others until now.

In 2023 I had an school internship at Mindmatters in Hamburg, which I really enjoyed. With my prior knowledge about VueJS and NuxtJS I designed and created my own little web app. In daily meetings I learned how to come from an idea to a final product. Also I was able to learn how their company works and how they manage everything.

Currently I mainly code in Java and VueJS with Tailwindcss, but still simple problems I want to solve in Python
